# 理工科复习台

这是一个本地优先的理工科期末复习助手，覆盖力学、数学、物理、化学、生物、电路、信号、控制、热工和计算机基础等课程。当前版本：`0.2.0`。

## 当前能力

- 按科目管理资料。
- 导入 `.pptx`、`.pdf`、`.txt`、`.md` 和常见图片文件。
- 支持直接粘贴纯文字例题导入，并在资料列表中继续修改题目文本。
- 从文本型 PPTX/PDF 中抽取文字，PPTX 会尽量保留 Office 公式对象、常见物理符号和幻灯片内嵌图片引用。
- 内置 89 条参考公式/递推式，覆盖工程力学、大学物理、化学、高等数学、生物学和算法复杂度，并保留适用条件、常见误用和开放参考来源。
- 每次导入或修改资料都会生成“增量知识包”，沉淀本资料的高优先知识点、公式、题型模板、防错点和训练模板，并进入后续知识地图、刷题、冲刺包和 API 上下文。
- 本地规则生成知识地图、复习提纲、题目、错题本和资料问答。
- 内置本地考试题型库和教材习题风格库，覆盖材料力学、理论力学、结构力学、弹性力学、流体力学、机械振动、工程力学综合、高等数学、大学物理、化学、生物学、电路分析、信号与系统、自动控制、热力学与传热学、数据结构与算法；刷题时不只依赖概念卡，也会补充常见大题、计算题、实验设计题、算法分析题和分步骤训练题。
- 按资料章节/专题、错题和已完成 sessions 生成期末复习计划，并推荐下一步复盘内容。
- 生成考前冲刺包，汇总科目下的资料、错题和复盘记录，输出优先专题、必背公式、易错清单和限时题。
- 可选填写 OpenAI-compatible API 配置，用内置 `ppt_zero_to_mastery_v1` Skill 生成“从 0 学会 PPT”的教学总结。
- 可选用内置 `stem_exam_teacher_v1` Skill 解题，输出分步讲解、关键公式、易错提醒、同类题提示和可固化复习记忆卡。
- API 模式下，图片文件和 PPTX 内嵌图片会直接作为模型视觉输入发送给兼容 API，不走本地图像识别。
- 设置页支持 API 模型检测、连接测试和 API Key 脱敏显示。
- 前后端做了索引和缓存优化，减少知识图谱、来源跳转、summary 排序和计划生成中的重复扫描。

## 启动

推荐日常使用双击启动：

- macOS App 图标：先运行 `npm run make:mac-app`，之后双击 `dist/理工科复习台.app`
- macOS：双击 `启动-理工科复习台.command`
- Windows：双击 `启动-理工科复习台.bat`

启动器会自动选择 `4173-4178` 之间可用的端口，并打开浏览器。如果复习台已经在运行，再次双击会直接打开已有页面，不会重复启动一份服务。

也可以在项目目录运行：

```bash
npm run open
```

如果想生成一个更像本地软件的 macOS 应用外壳：

```bash
npm run make:mac-app
```

生成后双击 `dist/理工科复习台.app` 即可，也可以把它拖到 Dock。这个 `.app` 带应用图标，启动失败时会弹出提示并把日志写入 `data/logs/launcher.log`。本地生成的 `.app` 会指向当前项目目录，所以不要随意移动本项目文件夹；如果移动了项目，重新运行一次 `npm run make:mac-app`。

常规开发启动：

```bash
npm install
npm start
```

然后手动打开：

```text
http://127.0.0.1:4173
```

如果当前终端没有可用的 `npm`，也可以直接运行：

```bash
/Users/shanfengye/.cache/codex-runtimes/codex-primary-runtime/dependencies/node/bin/node server.cjs
```

## 测试

无需启动网页服务的核心测试：

```bash
npm run test:core
```

离线准确率与来源引用检查：

```bash
npm run accuracy
```

启动服务后可运行接口烟测：

```bash
npm run smoke
```

## 打包给朋友体验

先在本机项目目录生成体验包：

```bash
npm install
npm run package:share
```

生成结果：

- `dist/stem-review-studio-share/`：可直接打开的体验包目录。
- `dist/stem-review-studio-share.zip`：可发给朋友的压缩包。

体验包会带上生产依赖、macOS `.app` 外壳和启动脚本，并使用一份空白 `data/db.json`，不会默认打包你当前导入的资料、错题、复习记录或 API Key。朋友电脑需要安装 Node.js 20 或更高版本；解压后 macOS 优先双击 `理工科复习台.app`，Windows 双击 `启动-理工科复习台.bat`。

## 数据位置

- 数据库：`data/db.json`
- 上传文件：`data/uploads/`

默认本地模式不会把资料发往外部服务。只有在设置里切换到 API 增强版并保存 API Key 后，生成总结、出题、问答或解题才会调用配置的 API；API Key 会写入用户目录中的独立密钥文件，前端状态中只显示 `__SET__`。

## 0.2.0 更新重点

- 新增 AI 教学 Skill：PPT 总结从“文字提取整理”升级为“从 0 学会并掌握”的教学路径。
- 新增 AI 解题 Skill：解题结果可直接保存为复习记忆，并能继续生成同类题提示。
- 支持 API 模式直接读取图片资料和 PPTX 内嵌图片，适合图片型课件；本地模式仍不做 OCR。
- 拆分 `lib/core/ai-skills.cjs` 和 `lib/server/api-teaching-materials.cjs`，减少 `server.cjs` 中的提示词和图片材料处理细节。
- 优化 `public/app.js` 来源定位和知识图谱连接渲染，优化 `server.cjs` summary/plan 路径的重复扫描。
- 启动与分享包统一为“理工科复习台”，macOS `.app` 带图标、日志和错误提示；旧“力学复习台”启动脚本已被替换。
